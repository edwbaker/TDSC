# Empty Bands Discovery

Identifies unused codes across multiple S-matrices. Unused bands can be
used to reduce the codebook as in Stammers (2011).

## Usage

``` r
emptyBands(...)
```

## Arguments

- ...:

  Two or more TDSC objects

## References

Stammers (2011) “Audio Event Classification for Urban Soundscape
Analysis”. PhD thesis. University of York.

## Examples

``` r
library(tuneR)
wave <- readWave(system.file("extdata", "1.wav", package="tdsc"))
t <- tdsc(wave)
emptyBands(t,t)
```
