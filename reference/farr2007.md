# Coding Matrix from Farr (2007)

Coding matrix used by Farr (2007).

## Usage

``` r
data(farr2007)
```

## Format

Matrix

## References

Farr (2007) “Automated Bioacoustic Identification of Statutory
Quarantined Insect Pests”. PhD thesis. University of Hull.

## Examples

``` r
library(tuneR)
wave <- readWave(system.file("extdata", "1.wav", package="tdsc"))
data(farr2007)
t <- tdsc(wave, coding_matrix=farr2007)
```
